# Homebrew Tap for Local AI Cat

Private, on-device AI chat and transcription for Mac.

## Install

```bash
brew tap local-ai-cat/tap
brew install --cask local-ai-cat
```

## Update

```bash
brew upgrade --cask local-ai-cat
```

Note: After initial install, the app updates itself via Sparkle (built-in auto-updater). Homebrew is primarily for the initial install.
